# our Portfolio

##  About The Project

This is my personal portfolio website, designed to showcase my skills, professional experience, and projects as a Full Stack Engineer. The application is built with a focus on performance, responsiveness, and user experience.

It features a modern design with **Dark/Light mode** support and **Bilingual content (English/Arabic)**, demonstrating my ability to build flexible and accessible web applications.

###  Key Features

* ** Modern Tech Stack:** Built with React.js and Vite for blazing fast performance.
* ** Responsive Design:** Fully responsive layout using Tailwind CSS.
* ** Dark & Light Mode:** Seamless theme switching with persistent state (Context API).
* ** Multi-language Support:** Full i18n support (English & Arabic) using React Context.
* ** Smooth Animations:** Interactive UI elements and scroll reveals.
* ** Mobile First:** Optimized for all screen sizes.

---

##  Technologies Used

* **Frontend:** [React.js](https://reactjs.org/), [Vite](https://vitejs.dev/)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **Icons:** [Lucide React](https://lucide.dev/)
* **State Management:** React Context API (for Theme & Language)
